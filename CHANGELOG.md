# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-29

### Added
- Complete fine-tuning course with 6 labs
- 11 example categories covering datasets, tokenization, training, and publishing
- Interactive demo with rich terminal output
- CI/CD pipeline with GitHub Actions
- Comprehensive test suite
- Pre-commit hooks for code quality
- PMAT compliance configuration

### Examples
- `loading/` - Dataset loading from CSV, JSON, Parquet, Hub
- `transform/` - Data transformations with map and filter
- `tokenize/` - Tokenization with padding and truncation
- `augment/` - Data augmentation techniques
- `imbalance/` - Handling imbalanced datasets
- `models/` - Working with pre-trained models
- `training/` - Trainer API usage
- `custom/` - Custom training configurations
- `callback/` - Training callbacks and logging
- `inferencing/` - Running inference
- `publishing/` - Publishing to Hugging Face Hub
