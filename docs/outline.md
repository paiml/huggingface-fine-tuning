## Quick Reference

### Module 1 Topics
- Datasets library fundamentals
- Loading from Hub, CSV, JSON, Parquet
- Dataset structure and features
- Map, filter, and transform operations
- Batched processing
- Tokenization (padding, truncation)
- Custom dataset creation
- Data augmentation
- Handling imbalanced data

### Module 2 Topics
- Trainer API configuration
- TrainingArguments
- Model selection (BERT, RoBERTa, DeBERTa)
- Inference with trained models
- Evaluation metrics
- Optimizer selection (AdamW)
- Learning rate schedules
- Mixed precision (FP16) training
- Custom callbacks
- TensorBoard/WandB logging
- Debugging training issues

### Module 3 Topics
- Hugging Face Hub overview
- Authentication and tokens
- push_to_hub method
- Model cards
- GitHub Actions
- Automated training pipelines
- Docker containers
- GitHub Container Registry

## Prerequisites

- Python 3.10+
- Basic understanding of machine learning concepts
- Familiarity with PyTorch (helpful but not required)

```bash
pip install transformers datasets huggingface_hub torch pandas scikit-learn
```
